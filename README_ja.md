Linux のユーザーおよびグループ管理、ファイル所有権、権限を習得します。シンボリックモードとオクタルモードを使用したファイルとディレクトリへのアクセス制御方法、sudo 権限の理解、umask によるデフォルト権限の設定方法を学びます。
